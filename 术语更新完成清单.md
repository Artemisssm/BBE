## 🎯 新的术语体系

### 核心概念

```
链路模式（8种）
  └─ 单元类型（4种）
      └─ 参数（多个）
```

### 8种链路模式

1. 返向中低速数传
2. 返向高速数传
3. ACM数传
4. 前向数传小环
5. 低速模拟源
6. 高速模拟源
7. ACM模拟源
8. 前向数传

### 4种单元类型

1. 编码单元 (ENCODE)
2. 调制单元 (MODULATE)
3. 解调单元 (DEMODULATE)
4. 译码单元 (DECODE)

## 📝 术语对比

| 层次 | 旧术语 | 新术语 | 英文 | 数量 |
|------|--------|--------|------|------|
| 第一层 | 单元名称 | **链路模式** | Link Mode | 8种 |
| 第二层 | 单元类型 | 单元类型 | Unit Type | 4种 |
| 第三层 | 参数 | 参数 | Parameter | 多个 |

## ✨ 优势总结

### 1. 避免混淆
- ❌ 旧：单元名称 vs 单元类型（都有"单元"）
- ✅ 新：链路模式 vs 单元类型（清晰区分）

### 2. 更专业
- ❌ 旧："单元名称"不是标准术语
- ✅ 新："链路模式"是通信领域标准术语

### 3. 层次清晰
```
链路模式（工作模式）
  └─ 单元类型（功能模块）
      └─ 参数（配置属性）
```

### 4. 易于理解
- 新用户更容易理解"链路模式"和"单元类型"的区别
- 减少学习成本
